## Full Description

```text
WearEdge Pro packages wearable frontline manufacturing evidence into an
OPEA-aligned five-agent suite.

Routes shown:
- maintenance -> CMMS work-order draft
- iqc -> QMS quality event
- changeover -> MES/checklist verification
- wi -> released work-instruction guidance
- hazard -> EHS observation

Repository:
https://github.com/davidmillerak2026-sys/wearedge-opea-manufacturing

OPEA upstream PR:
https://github.com/opea-project/GenAIExamples/pull/2462

Evidence highlights:
- Docker-runnable Web/API demo console
- Qdrant RAG and route-specific knowledge sources
- OPEA-compatible /v1/embeddings profile
- Official OPEA TEI profile with Hugging Face TEI and opea/embedding
- GenAIEval-compatible 15-case route evaluation
- Google Cloud C3 Xeon AVX-512/AMX hardware evidence

Safety boundary:
This is an assistive industrial decision-support prototype. It does not grant
restart permission, quality release, maintenance release, safety clearance, or
final root cause without human confirmation.
```
