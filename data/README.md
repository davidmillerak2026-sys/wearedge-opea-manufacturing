# Data Fixtures

This directory contains small, sanitized fixtures used by the public OPEA
public package. They are intentionally reviewable so evaluators can reproduce
RAG retrieval, route evaluation, action-card generation, guardrails, and
scorecard behavior.

The broader WearEdge program has private enterprise production-data lineage,
including quality-inspection work such as toothbrush workshop visual inspection
for IQC/OQC-style defect detection. Raw customer plant images, labels, lot
identifiers, operator identifiers, customer names, and production records are
not committed to this repository.

Keep only privacy-safe fixtures here.
